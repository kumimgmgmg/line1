#路線->駅検索プログラム
### 概要
+ 複数の路線から共通する駅を検索するプログラム
+ フォームは6つまで追加可能
+ フォームの入力候補をJavascriptで一覧表示

### 仕様
+ HTML5
+ CSS3
+ Javascript(Jquery2系)
+ Jinja2(HTMLテンプレート)

+ 駅データ.JP - 路線API(http://www.ekidata.jp/api/api_line.php)
+ Python(3.5)
+ Flask(Webアプリケーションフレームワーク)
+ PostgreSQL
